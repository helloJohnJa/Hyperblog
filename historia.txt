Poema al programador (tomado de internet)

Por ti seré capaz de convertirme en programador o quizá en hacker,
Usare el visual basic para crear un software ejecutable,
Que ingrese a tu sistema operativo y sea estable,
Como precaución pondré un antivirus con todo y parche creado por un cracker.

Usare mi poco conocimiento de informática,
Empezare ejecutando el comando suprimir,
borrara los malos recuerdos y dibujara una sonrisa en tu rostro para imprimir,
Después instalare el software que usa códigos ASCII, aritmética y lógica.

Creare un software que no sea complicado,
Que limpie tu memoria ram para quitar errores,
Para que nunca mantengas enojos y rencores,
Este software no lo conseguirás en ningún mercado.

Este comando funcionara con comandos de tristeza o ternura,
Depende del estado de animo ejemplo: help me, i’m sad o i’m happy, etcétera,
Son comandos sencillos que los guardaras en tu cabecera,
Los cuidaras tanto como si fuera una criatura.

Este software lo podes llevar en una computadora portátil,
Es un archivo autoextraíble,
Es totalmente increíble,
Porque es un programa volátil.

Este software del amor siempre estará disponible,
Solo tienes que abrir el window de tu alma para que cargue,
Lo podrás usar en la noche, en la mañana o en la tarde,
como medida de precaución no lo trates mal porque es auto destructible

úsalo cuando tengas sentimientos de inferioridad,
si tienes dudas conéctate a mi servidor,
llamado “El loco poeta soñador”,

Autor: desconocido
